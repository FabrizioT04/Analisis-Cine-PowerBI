Este proyecto es un simulador analítico desarrollado para evaluar la eficiencia operativa de una cartelera de cine
El objetivo principal es identificar patrones de consumo, horas pico de asistencia y la rentabilidad de las funciones programadas, permitiendo una mejor toma de decisiones comerciales
Herramientas Utilizadas:
*Excel: Procesamiento de datos, cálculo de KPIs (Ocupación y Alerta de Demanda) y limpieza inicial
*Power BI: Creación de dashboards interactivos para la visualización de datos y análisis de tendencias
*Funcionalidades del Dashboard
*KPIs Operativos: Visualización clara de Ingresos Totales por película y Porcentaje de Ocupación por sala
*Segmentación Temporal: Filtros interactivos por fecha para analizar el comportamiento diario
*Insights de Negocio: Comparativa de género favorito vs. rendimiento en taquilla, identificando oportunidades para optimizar la programación de salas
